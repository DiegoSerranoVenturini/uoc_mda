def divide_target(df, col, target_col):
    pos = df[df[target_col] == 1][col]
    neg = df[df[target_col] == 0][col]
    return pos, neg

- Qu mtodo de reduccin de dimensionalidad ha resultado ms efectivo para visualizar las dos clases de diagnstico de forma separada?
- BONUS: cundo es ms apropriado usar PCA o t-SNE?
  • Los dos metodos diferencian bastante bien las dos clases. Afinando mucho el comentrio podriamos decir que el metodo PCA genera una division lineal muy buena. En la frontera que provoca existe cierta confusion pero son casos que estan muy cerca los unos de los otros. Sin embargo, la frontera de t-SNE es menos "limpia" teniendo falsos negativos claros.
  • Por regla general, en cuando a tecnicas descritivas, t-SNE es mejor cuando el numero de variables es alto, y las relaciones pueden ser no-lineales; siendo mas practico y efectivo PCA cuando las relaciones subyacentes son lineales y se requiere menor tiempo de procesamiento (para datasets grandes tSNE puede tardar mucho tiempo en calcularse). A efectos de empleabilidad, solo PCA se puede emplear como tecnica predictiva (de hecho, hasta el objeto de sklearn de tSNE carece del metodo transform necesario para la fase de prediccion de un modelo).
